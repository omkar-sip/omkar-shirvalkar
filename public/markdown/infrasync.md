# InfraSync

**InfraSync** is a Digital Management System for Government Construction Projects, designed to bring transparency, accountability, and efficiency to public infrastructure development.

## Key Features

- 🏗️ **Project Tracking** – Real-time status of ongoing construction projects
- 📋 **Task Management** – Assign and monitor tasks across departments
- 📊 **Progress Reports** – Automated reporting and milestone tracking
- 🔐 **Role-Based Access** – Separate portals for engineers, contractors, and officials
- 📍 **Site Geo-tagging** – Pinpoint project locations on an interactive map
- 📷 **Photo Evidence** – Upload site photos to verify work completion

## Tech Stack

- **Frontend:** React + TypeScript
- **Backend:** Firebase / Node.js
- **Maps:** Leaflet
- **State Management:** Zustand
