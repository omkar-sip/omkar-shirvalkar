# HomieSearch

**HomieSearch** is a smart rental and PG (Paying Guest) discovery platform that helps students and working professionals find the best accommodation deals near them.

## Key Features

- 🏠 **Smart Search** – Filter by budget, location, amenities, and room type
- 📍 **Map Integration** – Browse listings on an interactive map
- 💬 **Direct Connect** – Contact landlords and PG owners directly
- ⭐ **Reviews & Ratings** – Authentic reviews from verified tenants
- 📱 **Mobile-First Design** – Optimized for on-the-go browsing

## Tech Stack

- **Frontend:** React + TypeScript
- **Maps:** Leaflet / Google Maps API
- **Backend:** Firebase / Node.js
