# Vaani AI

**Vaani AI** is a voice-powered AI assistant that enables seamless, natural conversations. Built to bridge the language gap, Vaani understands and responds in multiple Indian languages.

## Key Features

- 🎙️ **Voice Interaction** – Speak naturally and get intelligent responses
- 🌐 **Multi-language Support** – Supports Hindi, Marathi, and English
- 🤖 **AI-Powered** – Built on top of cutting-edge language models
- ⚡ **Real-time Processing** – Low latency voice recognition and synthesis
- 📱 **Responsive Design** – Works seamlessly on mobile and desktop

## Tech Stack

- **Frontend:** React / TypeScript
- **AI:** Gemini API / OpenAI Whisper
- **Voice:** Web Speech API
