# VRNN Website

**VRNN** (Vijay Raghavendra Nirmiti Nidhi) is a student innovation and entrepreneurship network. This is the official website connecting students, innovators, and creators across colleges.

## Key Features

- 🌐 **Community Hub** – Central platform for VRNN members
- 📰 **News & Updates** – Latest events, hackathons, and opportunities
- 🎓 **Student Showcase** – Highlight student projects and achievements
- 📞 **Connect** – Reach out to mentors and fellow innovators

## Tech Stack

- **Frontend:** HTML / CSS / JavaScript
- **Hosting:** GitHub Pages / Vercel
